# 进化模拟经营游戏设计文档

## 游戏概述

**游戏名称：** 进化纪元 (Evolution Era)
**类型：** 模拟经营 + 进化模拟
**核心体验：** 从单细胞到精神能量体的史诗进化之旅，每个阶段都有独特机制

---

## 核心设计理念

### 1. 阶段独特性原则
每个进化阶段必须有完全不同的核心机制，避免重复感：
- 单细胞：物理碰撞 + 吞噬
- 多细胞：细胞分化 + 信号传导
- 海洋生物：群体AI + 水流模拟
- 陆地生物：工具链 + 领地意识
- 原始人类：社会分工 + 知识传承
- 古代文明：贸易网络 + 军事征服
- 工业时代：流水线 + 能源管理
- 信息时代：算法优化 + 网络效应
- 太空文明：星际物流 + 资源开采
- 宇宙文明：时间操控 + 维度折叠
- 精神能量体：意识创造 + 现实改写

### 2. 自动化继承系统
每个阶段的自动化产出会转化为下一阶段的"基础加成"：
- 单细胞分裂速度 → 多细胞生长速度加成
- 海洋生物群体规模 → 陆地生物社会复杂度
- 工业产能 → 信息时代算力基础
- 星际殖民地数量 → 宇宙文明时间线分支

### 3. 数字飞升曲线
```
单细胞：10^0 → 10^3 (千级)
多细胞：10^3 → 10^6 (百万级)
海洋生物：10^6 → 10^9 (十亿级)
陆地生物：10^9 → 10^12 (万亿级)
原始人类：10^12 → 10^15 (千万亿)
古代文明：10^15 → 10^18 (百京)
工业时代：10^18 → 10^21 (十垓)
信息时代：10^21 → 10^24 (秭级)
太空文明：10^24 → 10^27 (千秭)
宇宙文明：10^27 → 10^30 (沟级)
精神能量体：∞ (无限)
```

## 阶段详细设计

### 阶段1：单细胞 (Cellular)

#### 主题
微观世界，原始海洋，生存与分裂

#### 核心机制
**吞噬系统**
- 玩家控制一个圆形细胞在2D平面移动
- 吞噬更小的营养颗粒成长
- 避开更大的捕食细胞
- 体积达到阈值后分裂

**分裂进化**
- 每次分裂可选择进化方向：
  - 鞭毛：移动速度+50%
  - 纤毛：转向灵活性+50%
  - 细胞壁：防御力+50%
  - 毒素囊：可释放毒素攻击

**环境模拟**
- 洋流：影响所有细胞移动
- 温度梯度：不同区域生长速度不同
- pH值：影响细胞健康

#### 资源
| 资源 | 说明 |
|-----|------|
| 营养颗粒 | 基础食物，随机生成 |
| 蛋白质 | 稀有资源，加速进化 |
| ATP | 能量货币，维持生命活动 |

#### 自动化
- **自动分裂**：达到一定大小自动分裂
- **趋化性**：自动向营养浓度高的区域移动
- **群体感应**：细胞间信号交流

#### 阶段目标
- 达到1000个细胞群体
- 解锁多细胞黏附能力

---

### 阶段2：多细胞 (Multicellular)

#### 主题
细胞协作，器官形成，从个体到组织

#### 核心机制
**细胞分化系统**
- 干细胞可分化成不同类型：
  - 表皮细胞：防御、感知
  - 肌肉细胞：运动
  - 神经细胞：信号传递
  - 消化细胞：营养吸收
  - 生殖细胞：繁殖

**信号传导网络**
- 神经细胞形成网络
- 信号从感知区域传导到反应区域
- 网络复杂度决定反应速度

**形态构建**
- 类似"接水管"游戏
- 连接不同类型的细胞形成器官
- 器官效率决定整体生存能力

#### 资源
| 资源 | 说明 |
|-----|------|
| 营养 | 维持细胞生存 |
| 氧气 | 细胞呼吸必需 |
| 分化因子 | 促进细胞分化 |

#### 自动化
- **自主节律**：心脏细胞自动搏动
- **反射弧**：简单的刺激-反应回路
- **激素系统**：化学信号自动调节

#### 阶段目标
- 形成完整消化系统
- 建立神经传导网络
- 达到100万细胞规模

---

### 阶段3：海洋生物 (Marine)

#### 主题
群体智能，鱼群算法，海洋生态

#### 核心机制
**群体行为AI**
- 基于Boids算法：
  - 分离：避免碰撞
  - 对齐：速度匹配
  - 聚合：向中心靠拢
- 群体规模越大，个体越安全

**生态位竞争**
- 不同物种占据不同生态位
- 食物链：浮游生物 → 小鱼 → 大鱼 → 顶级掠食者
- 共生关系：清洁鱼与大型鱼类

**迁徙系统**
- 季节性迁徙寻找食物
- 洄游繁殖
- 洋流利用

#### 资源
| 资源 | 说明 |
|-----|------|
| 浮游生物 | 基础食物 |
| 小鱼 | 中级食物 |
| 领地 | 繁殖和休息区域 |

#### 自动化
- **群体领袖**：自动寻找最优路径
- **警戒系统**：发现危险自动预警
- **繁殖仪式**：自动寻找配偶和产卵地

#### 阶段目标
- 群体规模达到10亿
- 建立跨洋迁徙路线
- 进化出初步的脊椎结构

---

### 阶段4：陆地生物 (Terrestrial)

#### 主题
征服陆地，工具使用，领地意识

#### 核心机制
**工具链系统**
- 收集材料制作工具：
  - 石器：砍伐、狩猎
  - 木器：容器、建筑
  - 骨器：缝纫、装饰
- 工具可升级，效率递增

**领地系统**
- 标记和保卫领地
- 领地内资源独享
- 领地大小决定群体规模上限

**巢穴建造**
- 类似"俄罗斯方块"的建造系统
- 不同房间有不同功能：
  - 储藏室：存储食物
  - 育儿室：提高幼崽存活率
  - 休息室：恢复体力

#### 资源
| 资源 | 说明 |
|-----|------|
| 食物 | 采集和狩猎获得 |
| 材料 | 石头、木头、骨头 |
| 领地标记 | 气味标记、声音标记 |

#### 自动化
- **哨兵系统**：自动警戒领地
- **幼崽看护**：轮流照顾后代
- **食物储藏**：自动保存多余食物

#### 阶段目标
- 建立永久巢穴
- 掌握火的使用
- 群体规模达到1万亿

---

### 阶段5：原始人类 (Primitive)

#### 主题
部落形成，农业革命，知识传承

#### 核心机制
**社会分工**
- 部落成员有不同职业：
  - 猎人：获取肉类
  - 采集者：获取植物
  - 工匠：制作工具
  - 萨满：知识传承、医疗
- 分工效率随人口增长而提升

**农业系统**
- 选择作物种植：
  - 谷物：高产但需精心照料
  - 根茎：耐储存
  - 果树：长期收益
- 季节循环影响收成

**知识传承**
- 长者向年轻人传授技能
- 技能等级影响生产效率
- 死亡导致知识流失

#### 资源
| 资源 | 说明 |
|-----|------|
| 食物 | 农业和狩猎产出 |
| 人口 | 部落规模 |
| 知识 | 技能和经验 |
| 信仰 | 凝聚力来源 |

#### 自动化
- **轮班制度**：自动安排工作和休息
- **贸易网络**：与其他部落交换物资
- **仪式系统**：定期举行增强凝聚力

#### 阶段目标
- 建立农业社会
- 发明文字记录知识
- 人口达到千万亿级

### 阶段6：古代文明 (Ancient)

#### 主题
城市兴起，帝国扩张，文明竞争

#### 核心机制
**城市建设**
- 六边形网格地图
- 建筑相邻加成：
  - 农田+水渠：产量+50%
  - 市场+道路：贸易+50%
  - 兵营+城墙：防御+50%

**贸易网络**
- 建立商路连接城市
- 贸易路线越长，利润越高
- 保护商路免受劫掠

**军事征服**
- 组建军队
- 战术选择：
  - 正面突击
  - 侧翼包抄
  - 围城战
- 占领城市获得资源

#### 资源
| 资源 | 说明 |
|-----|------|
| 粮食 | 维持人口 |
| 黄金 | 贸易和雇佣 |
| 石材 | 建筑 |
| 铁矿 | 武器 |
| 文化 | 解锁政策 |

#### 自动化
- **税收系统**：自动征收资源
- **官僚体系**：自动管理城市
- **常备军**：自动防御和巡逻

#### 阶段目标
- 建立帝国
- 统一大陆
- 达到百京级人口

机器轰鸣，能源革命，大规模生产

#### 核心机制
**工厂流水线**

- 瓶颈分析和优化

  - 石油：高效但有限
  - 电力：清洁但需要基础设施
- 电网平衡：供需匹配

**科技研发**
- 研究点数投入不同领域：
  - 机械：提高效率
  - 化学：新材料
  - 电力：新能源

#### 资源
| 资源 | 说明 |
|-----|------|
| 煤炭 | 基础能源 |
| 钢铁 | 工业基础 |
| 石油 | 高级能源 |
| 电力 | 清洁动力 |
| 科研点 | 技术进步 |

#### 自动化
- **自动采矿**：矿井自动产出
- **流水线**：全自动生产
- **电网**：自动分配电力

#### 阶段目标
- 实现全自动化生产
- 建立全球电网
- 产能达到十垓级

## 跨阶段系统

### 进化点数 (Evolution Points)
完成每个阶段目标获得进化点数，用于：
- 跨阶段加成（如：单细胞效率+10%）
- 解锁隐藏阶段（如：机械飞升、生物融合）
- 新游戏+模式

### 遗产系统
每次进化时，上一阶段的产出转化为遗产：
```
单细胞遗产 = 细胞数量 × 0.001 → 多细胞生长速度加成
海洋生物遗产 = 鱼群规模 × 0.0001 → 陆地生物社会复杂度
工业遗产 = 工厂产能 × 0.00001 → 信息时代算力基础
```

### 时间压缩
后期阶段时间流速加快：
- 单细胞：实时
- 多细胞：2x
- 海洋生物：5x
- ...
- 精神能量体：10000x

## 技术实现

### 架构设计
- 阶段系统：每个阶段独立模块
