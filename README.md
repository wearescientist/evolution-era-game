# 进化纪元 (Evolution Era)

从单细胞生物开始，经历11个进化阶段，最终成为精神能量体的史诗模拟经营游戏。

## 🎮 游戏特色

- **11个独特进化阶段**：每个阶段都有完全不同的核心机制
- **数字飞升体验**：从10^0到∞的指数级成长
- **自动化继承**：上一阶段的产出转化为下一阶段的基础加成
- **渐进式解锁**：科技树、升级系统、自动化逐步开放

| 2 | 多细胞 | 细胞分化、器官形成 | 1,000,000 |

## 🚀 快速开始

### 在线游玩
直接在浏览器中打开 `index.html` 即可开始游戏。

### 本地运行
```bash
cd evolution-game
python -m http.server 8080
# 或
npx serve .
```

## 📖 游戏指南

### 阶段1：单细胞

**操作方式：**
- 键盘：WASD 或方向键移动
- 鼠标/触摸：点击/触摸引导移动

**核心玩法：**
1. 控制细胞移动，吞噬绿色营养颗粒
2. 积累足够营养后细胞会变大
3. 达到分裂阈值后自动分裂
4. 收集金色蛋白质解锁进化升级

**升级系统：**
- 鞭毛：移动速度+50%
- 纤毛：转向灵活性+50%
- 细胞壁：防御力+50%
- 毒素囊：获得攻击能力
- 线粒体优化：能量产出+100%
- 自动分裂：开启自动模式
- 趋化性：自动寻找营养

**进化条件：**
- 细胞数量达到1000即可进化到下一阶段

### 通用操作

- **升级面板**：点击底部"升级"按钮打开
- **自动模式**：点击"自动"按钮开关自动寻路
- **进化按钮**：当满足条件时，底部会出现进化按钮
- **设置**：导出/导入存档、保存/加载游戏

## 💾 存档系统

- **自动保存**：每30秒自动保存
- **手动保存**：设置面板中点击"保存"
- **导出/导入**：将存档导出为JSON文件或从文件导入

## 🎯 进化点数

完成每个阶段目标获得进化点数，可用于：
- 跨阶段加成
- 解锁隐藏内容
- 新游戏+模式
